# 🔴 Moving Ball Game

A simple interactive game designed to practice event handling and coordinate boundary management in Pygame.

## Features
- **Responsive Movement**: The ball moves instantly in response to arrow key inputs.
- **Boundary Detection**: Prevents the ball from moving off-screen, maintaining it within the window dimensions.
- **Clean UI**: Minimalist design with a focus on smooth logic.

## Game Rules
- Use the **Arrow Keys** (Up, Down, Left, Right) to move.
- Each movement shifts the ball by 20 pixels.
- The ball is constrained to the screen size (e.g., 600x600 or 800x800).

## Requirements
- Python 3.x
- Pygame
